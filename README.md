A classic, minimalist 2D arcade game built to replicate the timeless retro Pong experience. 
Players control vertical paddles to deflect a ball back and forth, scoring points when the opponent fails to return the ball.
FeaturesSmooth Controls: Responsive paddle movement.Score Tracking:
Real-time score display for both players.
Adaptive Ball Physics: Increasing ball speed over time to heighten difficulty.
Retro Visuals: Clean, pixel-perfect minimalist aesthetic.

Game ModesSingle Player: 
Challenge an AI opponent with scaling difficulty.

Local Multiplayer:
Face off against a friend on the same keyboard.
